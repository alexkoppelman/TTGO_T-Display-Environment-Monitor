# TTGO_T-Display-Environment-Monitor

**Functionality:**
- Works online and Offline
- When online pushes data to MQTT 
- Buttons to Sleep and Reset/Start

**Parts List:**
- TTGO T-Display ESP32
- 1000mHa Battery
- BME280
- CCS811
